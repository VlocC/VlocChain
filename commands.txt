This is a key for using curl commands to work the program. Will be updated soon to not use curl

First run
$ python3 app.py

Generate a new user- 
$ curl "localhost:5000/newuser"

Emulate mining a "token"-
$ curl "localhost:5000/mine"

Emulate a Transaction-
$ curl "localhost:5000/print" -H "Content-Type: application/json" -d '{"sender":"YOURUSERNAME","reciever":"RECIEVER_USERNAME","fname":"MESSAGE"}'    


Print out messages that have been sent-
$ curl "localhost:5000/print" -H "Content-Type: application/json" -d '{"username":"YOURUSERNAME"}'
